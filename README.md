Evidencia de aprendizaje. Programa con hilos

Como parte de la evaluación de esta unidad, es necesario que realices un programa en JAVA 
que permita la manipulación de hilos. 
Propósito: Desarrollar un programa que permita ejecutar tareas con hilos. 
Instrucciones:

1.  Identifica los componentes y la estructura requerida para la manipulación de hilos en los 
ejemplos de códigos presentados en el desarrollo de la unidad 2, esto lo lograrás analizando 
dicho código presentado.

2.  Desarrolla la interfaz gráfica del programa, esta deberá contar con los componentes gráficos
necesarios para que el programa sea funcional.

3.  Desarrolla los métodos que proporcionen la funcionalidad requerida al programa.
